# Facial Emotion Detection

A deep learning based facial emotion detection system developed using Python, TensorFlow, OpenCV, and Dlib.

## Features
- Detects facial emotions in real-time
- Uses CNN model for prediction
- Dlib facial landmarks support
- Webcam-based emotion detection

## Technologies Used
- Python
- TensorFlow
- OpenCV
- Dlib
- NumPy

## Files Included
- emotion_detection.py
- dlib_landmarks.py
- CNN multiclass emotion detection task.ipynb

## Dataset
FER2013 dataset
